# Regla: UI/UX

## Uso
Aplicar a todo código de interfaz de usuario.

## Reglas
1. SIEMPRE usar Tailwind CSS — no CSS inline o modules
2. SIEMPRE usar español chileno en toda la UI
3. SIEMPRE ser responsive — mobile-first
4. SIEMPRE usar componentes reutilizables
5. SIEMPRE manejar estados vacíos y loading
6. SIEMPRE usar iconos consistentes (lucide-react)
7. SIEMPRE mantener contraste WCAG AA

## Paleta
- Primario: `#1e3a5f` (azul oscuro)
- Acento: `#c9a227` (dorado)
- Exito: `#2e7d32` (verde)
- Error: `#c62828` (rojo)
- Fondo: `#f5f5f0` (gris cálido)
