# Documentation Skill

Guía para crear documentación clara y completa.

## Tipos de Documentación

### Técnica
- README.md — Descripción del proyecto
- AGENTS.md — Agentes y convenciones
- API docs — Endpoints y uso
- Architecture docs — Diseño del sistema

### Administrativa
- Informes técnicos
- Actas de reuniones
- Oficios y comunicaciones
- Protocolos de actuación

## Convenciones

### Markdown
- Títulos con `#` jerárquicos
- Código con ```language
- Tablas para datos estructurados
- Listas para pasos
- Enlaces relativos

### Español Chileno
- Tuteo formal
- Voseo solo si el contexto lo requiere
- Sin anglicismos innecesarios
- Términos técnicos en español cuando existen

### Estructura de README
```markdown
# Nombre del Proyecto

Descripción breve.

## Instalación
## Uso
## Arquitectura
## Contribución
## Licencia
```
