# Academic Writing

Guía para redacción académica y científica.

## Formato APA 7ª Edición

### Estructura de Paper
1. Título
2. Resumen (máx. 250 palabras)
3. Palabras clave
4. Introducción
5. Marco Teórico
6. Metodología
7. Resultados
8. Discusión
9. Conclusiones
10. Referencias
11. Anexos

### Citas
- **Narrativa:** "Según Pérez (2023), ..."
- **Parentética:** (Pérez, 2023)
- **Múltiples:** (Pérez, 2023; González, 2022)
- **Direta:** "cita textual" (Pérez, 2023, p. 45)

### Referencias
- Libro: Autor, A. A. (Año). *Título*. Editorial.
- Artículo: Autor, A. A. (Año). Título. *Revista*, *vol*(núm), pp-pp.
- Web: Autor, A. A. (Año). Título. Sitio Web. URL

## Herramientas
- Google Scholar para buscar fuentes
- Zotero/Mendeley para gestión de referencias
- Grammarly para revisión (inglés)
- Corrector ortográfico para español
