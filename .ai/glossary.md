# Domain Glossary — Términos del Sistema

## Términos de Negocio (Español Chileno)

| Término | Definición |
|---------|------------|
| **Amonestación** | Carta formal que documenta faltas y advierte sobre consecuencias |
| **Anotación** | Registro individual de conducta (positiva, negativa o informativa) |
| **Apoderado** | Padre, madre o tutor legal del estudiante |
| **Bitácora** | Registro cronológico de acciones en un caso |
| **Carta de Compromiso** | Documento donde el apoderado se compromete a apoyar medidas correctivas |
| **Causa** | Expediente completo de un caso disciplinario |
| **Debido Proceso** | Conjunto de garantías procedimentales (notificación, descargos, apelación) |
| **Derivación** | Envío del caso a instancia superior (dirección, Superintendencia) |
| **Descargos** | Derecho del estudiante a ser escuchado y presentar su versión |
| **Entrevista Personal** | Reunión con el estudiante para recoger su versión |
| **Fase** | Etapa del proceso disciplinario (5 fases) |
| **Falta** | Conducta contraria al Reglamento Interno |
| **Medida Cautelar** | Acción preventiva mientras se investiga |
| **Medida Disciplinaria** | Sanción aplicada tras proceso (formativa o correctiva) |
| **Medida Formativa** | Acción pedagógica para reparar daño (prioritaria) |
| **Notificación** | Comunicación formal a apoderados sobre el proceso |
| **RICE** | Categorización de conductas según Reglamento Interno |
| **Suspensión** | Medida que impide asistir a clases (máx. 15 días hábiles) |

## Términos Técnicos

| Término | Definición |
|---------|------------|
| **Tenant** | Establecimiento educacional (cole, liceo) |
| **RLS** | Row Level Security — aislamiento de datos por tenant |
| **JWT** | JSON Web Token — sesión de autenticación |
| **RPC** | Stored Procedure o función en Supabase |
| **FSD** | Feature-Sliced Design — arquitectura del frontend |
| **Signed URL** | URL temporal para acceder a storage privado |
